This hands-on workshop was delivered as part of the **CoE MicroPlanet Global Ambassador Program (GAP)**, focusing on transparency, scripting, and interpretation in biological data analysis.

## Topics Covered

- Data wrangling with the `tidyverse` ecosystem
- Data visualization with `ggplot2`
- Reproducible reporting with R Markdown
- Shifting from click-based tools (e.g., SPSS) toward scripted, transparent workflows
- Practical exercises using biological datasets

It matters because undergraduate biology education at many Indonesian universities still relies heavily on click-based software, which obscures analytical logic and prevents students from understanding how results are constructed or reproduced. This workshop aimed to bridge that gap by introducing open-source, reproducible tools.
